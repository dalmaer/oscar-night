## Project Overview

Oscar Night is a collaborative voting application where friends can vote on Oscar nominations together.

## Project Status

This is a new project in early development. The technology stack has not yet been established, but the .gitignore indicates it will be a Node.js-based web application.

## Getting Started

When setting up this project, you'll need to:
1. Choose a frontend framework (React, Vue, Next.js, etc.)
2. Initialize with `npm init` or equivalent
3. Set up build configuration based on chosen framework
4. Create `.env` file from `.env.example` (when available)
